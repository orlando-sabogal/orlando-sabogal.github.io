Spatial Analysis Workshop in Uruguay - June 2019

This is a short course about using geographic data to generate maps and insights. We will see how to use the sf library to load and process data by performing spatial operations. For map-making we are going to introduce the tmap library. Also, we will see how to create interactive maps with leaflet. You can see all the content of the course
